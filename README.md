# Digitizing 3D Humans - Client

This is a client (front-end) for the project.

## Requirements:

* CMake (I've used 3.9.1)
* OpenCV (3.4)
* Freenect2 (0.2.0)
* GLFW (3.3.0)
* yaml-cpp (https://github.com/jbeder/yaml-cpp)
* rpc-lib (https://github.com/rpclib/rpclib)

## Compilation:

```
mkdir build && cd build
cmake ..
make
make install
```
